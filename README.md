# 🏥 MedLink

### Your integrated platform to find and reserve medicines easily and quickly.

![MedLink Logo](frontend/images/logo.png)

---

## 🌟 Overview
**MedLink** is a comprehensive web development project (part of the Distributed and Parallel Computing Laboratory - DS) designed to bridge the gap between patients and pharmacies. The platform solves the problem of manual and exhausting searches for medicines by providing a digital marketplace that tracks real-time availability and stock.

---

## 🚀 Key Features

### 👨‍👩‍👧‍👦 For Citizens
- **Smart Search:** Advanced search engine to find medicines by name or active ingredient.
- **Real-time Availability:** Instantly see which nearby pharmacies have the medicine in stock.
- **Reservation System:** Securely reserve your prescriptions online before visiting the pharmacy.
- **Favorites:** Save your most-used medicines and preferred pharmacies for quick access.

### 💊 For Pharmacies
- **Inventory Management:** Easy-to-use dashboard to update stock status in real-time.
- **Digital Presence:** Expand reach to local customers and reduce phone-based inquiries.
- **Analytics:** Track customer demand patterns and view stock-out reports.

### 🛡️ For Administrators
- **Network Oversight:** Verify and manage licensed pharmacies within the system.
- **Shortage Tracking:** Generate reports on medicine shortages across different regions.
- **Access Control:** Manage user permissions and ensure data security across the platform.

## 🛠️ Tech Stack

Built with a focus on speed, modern design, and performance:

- **Frontend:** Core HTML5, Modern CSS3 (Glassmorphism & Neumorphism design), Vanilla JavaScript.
- **Design System:** Vibrant UI with dynamic blobs, gradients, and micro-animations.
- **Icons:** [Font Awesome 6.4.0](https://fontawesome.com/)
- **Fonts:** Google Fonts ([Outfit](https://fonts.google.com/specimen/Outfit), [Inter](https://fonts.google.com/specimen/Inter))
- **Logic:** Custom-built data engine (`data-engine.js`) handling filtering, sorting, and dynamic pagination.

## 🏁 Getting Started

To run the project locally without any installation requirements:
1. Clone or download the repository.
2. Navigate to the `frontend` folder.
3. Open `index.html` in any modern web browser.
